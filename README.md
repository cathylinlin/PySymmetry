# PySyM

PySyM (Python Symmetry) 是一个用于描述对称性及其在物理学中应用的 Python 库。  
本项目**目前**已开发完成了核心数学层和物理抽象层。

## 项目简介

PySyM 旨在为物理学、数学和晶体学研究者提供一个强大而灵活的对称性分析工具。通过抽象的群论框架和丰富的具体群实现，用户可以轻松地研究各种对称性结构。

### 目前实现

- 常见代数结构
- 群论
- 李代数理论
- 表示论
- 线性代数
- 物理对象
- 物理表示
- 对称环境
- 对称操作
- 工具模块

### 技术特点

- 完全类型注解，提供良好的代码提示
- 遵循 Python 最佳实践和 PEP 8 规范
- 模块化设计，易于扩展新的群类型
- 高效的算法实现，支持大群的采样检查

### 设计哲学与 API 规范

本项目的所有模块（尤其是 `abstract_phys`）严格遵循**数学与物理的严格分离与映射**，并遵循 **PEP 8** 命名规范：
- **物理场 (Fields)**：采用函数式与几何学定义，核心参数为 `field_function`（不使用强绑定的 `mass` 或 `spin` 属性），计算方法统一规范为 `.evaluate(position)` 和 `.gradient()`，不仅能满足泛函数学表达，也有着更加精准的代码语义。
- **对称操作 (Operations)**：如 `ParityOperation`, `TimeReversalOperation` 等均实现 `.act_on(obj)` 方法，通过鸭子类型 (Duck typing) 检查待作用对象的属性 (如 `position`, `velocity`) 来进行物理量变换。
- **测试用例一致性**：测试套件（如 `test_abstract_phys_robust.py` 等）已同核心 API 同步更新，所有的测试断言均匹配真实函数的命名规范并取消了防御性的 Skip。
